<h3>源码简介：</h3>
<p>安装说明：直接放入服务器或者空间，访问域名根据安装向导进行安装。
程序魅力：此程序是类似微信小程序一样的机制系统，但不是微信小程序，跟微信不搭界。
此程序可以安装N个小程序后，可以同时访问N个小程序，且数据可互通，可不互通，由开发者定义。<p>
<p>程序提供插件机制，方便开发者进行扩展。官网www.tinkphp.cn，可以按照开发手册设计自己的喜欢的小程序。<p>
